## 🎯 About
 
JS Calculator is a fully functional calculator running entirely in the browser. It supports chained arithmetic operations, decimal input, and three distinct clear modes. The entire logic is encapsulated in a `Calculator` class, making it a practical exercise in Object-Oriented JavaScript applied to a real interactive UI.
 
## ✨ Features
 
- ➕ **Arithmetic Operations** - Addition, subtraction, multiplication, and division
- 🔗 **Chained Calculations** - Perform sequential operations without resetting
- 🔢 **Decimal Support** - Dot input with duplicate-dot prevention
- 🗑️ **Three Clear Modes** - `DEL` removes the last digit, `CE` clears current input, `C` resets everything
- 🖥️ **Dual Display** - Separate lines for the previous operation and current input
 
## 🛠️ Built With
 
- **HTML5** - Semantic button grid and display containers
- **CSS3** - CSS Grid for the button layout, Flexbox for centering, and hover state styling
- **JavaScript** - `Calculator` class with OOP encapsulation, DOM manipulation, and event delegation
 
## 🚀 Getting Started
 
### Prerequisites
 
- A modern web browser (Chrome, Firefox, Safari, Edge)
 
### Installation
 
```bash
# Clone the repository
git clone https://github.com/Luan-Neumann-Dev/js-calculator.git
 
# Navigate to project directory
cd js-calculator
 
# Open in browser
open index.html
```
 
## 📁 Project Structure
 
```
js-calculator/
├── index.html          # Calculator markup — display and button grid
├── css/
│   └── style.css       # Grid layout, button styles, and display formatting
└── js/
    └── script.js       # Calculator class — all logic, state, and DOM updates
```
 
## 🎓 What I Learned
 
- Applying Object-Oriented Programming in JavaScript with classes and constructors
- Managing application state through class properties instead of scattered global variables
- Building a button grid layout using CSS Grid with `grid-column: span`
- Handling edge cases such as duplicate dots, empty inputs, and sequential operator changes
